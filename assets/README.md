# Assets

Icons for the solar forecasting tool:

- **automation-bash-icon.png** – Terminal/automation style (dark background, teal accent). Use for generic automation or DevOps.
- **solar-automation-icon.png** – Solar-themed (sun + terminal, amber/gold on dark blue). Use for the `run.sh` / solar forecasting launcher.

To use as the icon for `run.sh` (Windows): create a shortcut to `run.sh`, right-click the shortcut → **Properties** → **Change Icon**, then browse to the PNG (or use an `.ico` converted from the PNG). On Linux/macOS you can use the PNG in desktop entries or launchers.
